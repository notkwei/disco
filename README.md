# Welcome to Disco!

Disco is a fast, configurable Python program that generates Minecraft resource and data packs that adds music discs to the game. 

## Installation
Disco requires Python 3.8 or higher, and also requires `rich`, `mutagen`, and `questionary`. 

### Linux
Clone the repo and install dependencies:
```
git clone https://github.com/notkwei/disco
cd disco
python -m pip install -r requirements.txt
```
Next, run Disco.
```
python main.py
```
